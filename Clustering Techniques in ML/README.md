# Clustering Algorithms Comparison

## Overview

This project explores and compares three popular clustering algorithms used in Unsupervised Machine Learning:

* K-Means Clustering
* Hierarchical Clustering
* DBSCAN (Density-Based Spatial Clustering of Applications with Noise)

The project demonstrates the implementation of K-Means clustering and compares its performance with Hierarchical Clustering and DBSCAN using the Iris dataset.

---

## Objectives

* Understand the fundamentals of clustering techniques.
* Implement K-Means clustering on a dataset.
* Apply Hierarchical Clustering and DBSCAN for comparison.
* Visualize the clusters generated by each algorithm.
* Evaluate clustering performance using Silhouette Score.

---

## Dataset

The project uses the **Iris** dataset available in Scikit-learn.

**Dataset Features:**

* Sepal Length
* Sepal Width
* Petal Length
* Petal Width

Number of Samples: 150

## Project Workflow

### 1. Data Loading

* Load the Iris dataset from Scikit-learn.

### 2. Data Preprocessing

* Standardize the dataset using StandardScaler.

### 3. K-Means Clustering

* Apply K-Means clustering.
* Generate cluster labels.
* Visualize the resulting clusters.

### 4. Hierarchical Clustering

* Apply Agglomerative Hierarchical Clustering.
* Generate cluster labels.
* Visualize the resulting clusters.

### 5. DBSCAN

* Apply DBSCAN clustering.
* Detect clusters based on density.
* Visualize the resulting clusters.

### 6. Performance Comparison

* Evaluate each clustering algorithm using Silhouette Score.
* Compare clustering effectiveness.

---

## Results

The clustering algorithms were compared using Silhouette Score, a metric that measures how well data points fit within their assigned clusters.

| Algorithm               | Clustering Type |
| ----------------------- | --------------- |
| K-Means                 | Centroid-Based  |
| Hierarchical Clustering | Hierarchy-Based |
| DBSCAN                  | Density-Based   |
